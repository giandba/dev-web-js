Variáveis

Uma variável armazena um conteúdo durante o tempo de execução do sistema. Seu tempo de vida depende de como e onde foi declarada

Toda variável possui, nome, tipo e conteudo. No caso do javascript, uma linguagem 'fracamente tipada', onde o tipo é definido pelo conteúdo.

No javascript temos 3 formas de declarar uma variável:

  • var: declara uma variável para receber qualquer conteúdo
  • let: declara uma variável para receber qualquer conteúdo, LOCAL, ESCOPO DE BLOCO
  • const: declara uma variável para receber qualquer conteúdo, SOMENTE LEITURA

Conteúdo pode ser de 7 tipos:

  • boolean: true ou fase
  • null: conteúdo nulo
  • undefined: conteúdo não definido
  • number: números inteiros ou com ponto flutuatne. Ex 123 ou 1.23
  • string: "texto"
  • object: qualquer coisa diferente do que temos acima
  • *symbol: cria uma instancia única (ecmascript 6)

Declarando variáveis

var nomeCompleto = "Giancarlo"

var periodoCurso = 1

var matriculaAtiva = true
